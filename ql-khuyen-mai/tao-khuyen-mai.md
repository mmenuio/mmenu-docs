# Tạo chương trình khuyến mại

## Các loại khuyến mại

| Loại | Ví dụ |
|------|-------|
| **Giảm % hóa đơn** | Giảm 10% tổng bill |
| **Giảm tiền cố định** | Giảm 50.000đ |
| **Mua X tặng Y** | Mua 2 ly tặng 1 ly |
| **Giảm theo món** | Giảm 20% món cụ thể |
| **Combo** | Gộp nhiều món với giá ưu đãi |

## Tạo chương trình mới

1. Nhấn **+ Tạo khuyến mại**
2. Chọn loại khuyến mại
3. Điền thông tin:

![Form tạo KM](../assets/images/vi/ql-khuyen-mai/02-tao-khuyen-mai.png)

| Trường | Mô tả |
|--------|-------|
| Tên chương trình | Tên hiển thị |
| Mức giảm | Số % hoặc số tiền |
| Điều kiện | Áp dụng khi tổng bill >= X |
| Thời gian | Ngày bắt đầu — kết thúc |
| Giờ áp dụng | Khung giờ cụ thể (tùy chọn) |
| Ngày trong tuần | Áp dụng ngày nào trong tuần |

4. Nhấn **Lưu** để kích hoạt
