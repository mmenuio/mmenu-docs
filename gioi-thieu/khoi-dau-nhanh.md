# Khởi đầu nhanh (Quick Start)

Hướng dẫn này giúp bạn bắt đầu sử dụng Mmenu Admin trong vòng 5 phút.

## Bước 1: Đăng nhập

1. Mở app Mmenu Admin trên thiết bị của bạn
2. Nhập số điện thoại và mật khẩu đã được cấp
3. Nhấn **Đăng nhập**

## Bước 2: Chọn cửa hàng

Sau khi đăng nhập, màn hình sẽ hiển thị danh sách cửa hàng bạn được phân quyền quản lý.

1. Tìm kiếm cửa hàng bằng ô **Tìm kiếm** ở trên
2. Nhấn vào tên cửa hàng để vào màn hình quản lý

> 💡 Nếu bạn quản lý nhiều chuỗi, màn hình **Tổng quan** sẽ hiển thị doanh thu tổng hợp của tất cả cửa hàng.

## Bước 3: Làm quen với giao diện chính

Sau khi chọn cửa hàng, bạn sẽ thấy màn hình chính với 12 module:

| Module | Chức năng |
|---|---|
| QL Order | Quản lý bàn và nhận order |
| Bếp/Bar | Màn hình chế biến món |
| QL Thực đơn | Quản lý menu sản phẩm |
| Báo cáo KD | Xem thống kê kinh doanh |
| QL Nhân viên | Quản lý tài khoản nhân viên |
| QL Ca | Mở/đóng ca làm việc |
| Vật tư | Quản lý thiết bị |
| QL Khuyến mại | Tạo chương trình ưu đãi |
| Sổ thu chi | Quản lý dòng tiền |
| Cài đặt | Thiết lập hệ thống |
| Quản lý công việc | Giao việc nội bộ |
| Yêu cầu hỗ trợ | Liên hệ đội ngũ Mmenu |

## Bước 4: Bắt đầu nhận order

1. Nhấn vào **QL Order**
2. Chọn khu vực → chọn bàn
3. Nhấn **+** để tạo order mới
4. Chọn món từ thực đơn → nhấn **Đặt món**
