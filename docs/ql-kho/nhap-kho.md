# Nhập kho

## Tạo phiếu nhập kho

1. Trong màn hình QL Kho, nhấn **+ Nhập kho**
2. Chọn hoặc tìm kiếm nguyên vật liệu
3. Nhập số lượng và đơn vị tính
4. Nhập giá nhập (tùy chọn)
5. Chọn nhà cung cấp (tùy chọn)
6. Nhấn **Lưu**

![Nhập kho](/img/vi/ql-kho/02-nhap-kho.png)

## Nhập kho hàng loạt

Nhấn **Nhập file Excel** để nhập nhiều nguyên liệu cùng lúc theo file mẫu.

## Lịch sử nhập kho

Tab **Lịch sử** hiển thị tất cả phiếu nhập, có thể lọc theo ngày, nguyên liệu hoặc nhà cung cấp.
