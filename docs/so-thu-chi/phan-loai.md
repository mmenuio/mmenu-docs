# Phân loại thu chi

**Danh mục thu chi** giúp phân loại các khoản vào/ra để dễ dàng phân tích chi phí vận hành.

![Phân loại thu chi](/img/vi/so-thu-chi/03-phan-loai.png)

## Danh mục chi phí mặc định

| Nhóm | Ví dụ |
|------|-------|
| Nguyên liệu | Mua thực phẩm, đồ uống |
| Nhân sự | Lương, thưởng nhân viên |
| Vận hành | Điện, nước, gas |
| Marketing | In ấn, quảng cáo |
| Thiết bị | Sửa chữa, mua mới |
| Khác | Các chi phí phát sinh |

## Thêm danh mục tùy chỉnh

1. Vào **Cài đặt → Sổ thu chi**
2. Nhấn **+ Thêm danh mục**
3. Đặt tên và chọn loại (Thu / Chi)
4. Nhấn **Lưu**
