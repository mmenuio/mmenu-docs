# Nhập file sản phẩm

Tính năng **nhập file** cho phép thêm hàng loạt sản phẩm bằng cách tải lên file Excel, tiết kiệm thời gian khi setup menu mới.

## Tải file mẫu

1. Trong màn hình QL Thực đơn, nhấn **Nhập file**
2. Nhấn **Tải file mẫu** để tải template Excel

![Nhập file](/img/vi/ql-thuc-don/05-nhap-file.png)

## Cấu trúc file mẫu

| Cột | Nội dung |
|-----|----------|
| Tên sản phẩm | Tên món |
| Danh mục | Tên danh mục (phải khớp với danh mục đã có) |
| Giá bán | Số nguyên, không có dấu phẩy |
| Đơn vị | Phần / Ly / Đĩa... |
| Mô tả | Tùy chọn |

## Tải lên file

1. Điền đầy đủ thông tin vào file mẫu
2. Nhấn **Chọn file** và chọn file Excel đã điền
3. Nhấn **Nhập** để xử lý
4. Kiểm tra kết quả nhập — hệ thống hiển thị số dòng thành công và lỗi

> Nếu có lỗi, tải về file báo lỗi để xem dòng nào cần sửa.
