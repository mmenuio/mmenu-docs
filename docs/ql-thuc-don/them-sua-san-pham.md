# Thêm và chỉnh sửa sản phẩm

## Thêm sản phẩm mới

1. Trong tab **Sản phẩm**, nhấn nút **+ Thêm** ở góc trên phải
2. Điền thông tin sản phẩm

![Form thêm sản phẩm](/img/vi/ql-thuc-don/02-them-san-pham.png)

## Thông tin sản phẩm

| Trường | Bắt buộc | Mô tả |
|--------|----------|-------|
| **Tên sản phẩm** | ✓ | Tên hiển thị trên menu và order |
| **Danh mục** | ✓ | Nhóm danh mục thuộc về |
| **Giá bán** | ✓ | Giá mặc định |
| **Hình ảnh** | | Ảnh minh họa món |
| **Mô tả** | | Mô tả ngắn cho khách |
| **Đơn vị** | | Phần, ly, đĩa... |
| **Trạm in** | | Bếp nóng / Bếp lạnh / Bar |
| **Tùy chọn** | | Gắn nhóm tùy chọn (topping, size...) |

3. Nhấn **Lưu** để tạo sản phẩm

## Chỉnh sửa sản phẩm

1. Nhấn vào tên sản phẩm trong danh sách
2. Chỉnh sửa thông tin cần thay đổi
3. Nhấn **Lưu**

![Chỉnh sửa sản phẩm](/img/vi/ql-thuc-don/03-sua-san-pham.png)
