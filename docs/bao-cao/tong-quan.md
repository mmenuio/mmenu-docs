# Báo cáo tổng quan

Module **Báo cáo KD** cung cấp hệ thống thống kê đa chiều giúp chủ cửa hàng và quản lý nắm bắt toàn diện hoạt động kinh doanh.

![Màn hình Báo cáo](/img/vi/bao-cao/01-tong-quan.png)

## Danh sách báo cáo

| Báo cáo | Mô tả |
|---------|-------|
| Doanh thu | Tổng quan doanh thu theo kỳ |
| Khách hàng | Thống kê lượt khách |
| Sản phẩm | Món bán chạy, số lượng |
| Món ăn ABC | Phân tích ABC theo doanh thu |
| Theo đơn | Chi tiết từng order |
| Thanh toán | Theo phương thức thanh toán |
| Nhân viên | Doanh thu theo từng nhân viên |
| Theo bàn | Doanh thu theo bàn |
| Theo khu vực | Doanh thu theo khu vực |
| Theo ca | Doanh thu theo ca làm việc |
| Đánh giá | Điểm đánh giá của khách |
| Thẻ thành viên | Doanh thu từ thẻ |
| Trả món | Thống kê món bị trả |
| Theo giờ | Phân bổ doanh thu theo giờ |
| Điều chỉnh hoá đơn | Lịch sử chỉnh sửa bill |
| Đối tác giao hàng | Doanh thu qua app giao hàng |
| Đặt chỗ | Thống kê đặt chỗ |
| TIP | Thống kê tiền tip |
| Lượt thích | Báo cáo món được yêu thích |
| Nguồn khách hàng | Kênh dẫn khách |
| Nhóm khách hàng | Phân tích theo nhóm KH |

## Bộ lọc chung

Hầu hết các báo cáo đều có bộ lọc:
- **Khoảng thời gian**: Hôm nay, tuần, tháng, tùy chỉnh
- **Cửa hàng**: Lọc theo chi nhánh (với tài khoản chuỗi)
