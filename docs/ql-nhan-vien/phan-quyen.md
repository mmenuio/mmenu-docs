# Phân quyền nhân viên

Mmenu Admin hỗ trợ hệ thống phân quyền theo **vai trò** — mỗi vai trò có các quyền truy cập chức năng khác nhau.

![Màn hình phân quyền](/img/vi/ql-nhan-vien/03-phan-quyen.png)

## Các vai trò mặc định

| Vai trò | Quyền chính |
|---------|-------------|
| **Quản lý** | Toàn quyền truy cập tất cả module |
| **Thu ngân** | QL Order, thanh toán, xem báo cáo cơ bản |
| **Bếp/Bar** | Chỉ xem màn hình Bếp/Bar |
| **Phục vụ** | Xem order, xác nhận phục vụ |

## Tùy chỉnh quyền

Ngoài các vai trò mặc định, quản lý có thể tạo vai trò tùy chỉnh:
1. Nhấn **+ Thêm vai trò**
2. Đặt tên vai trò
3. Tick chọn từng quyền theo module
4. Nhấn **Lưu**

## Gán vai trò cho nhân viên

Trong form chỉnh sửa nhân viên, chọn **Vai trò** từ dropdown để thay đổi phân quyền.
