# Tạo đặt chỗ mới

## Các bước tạo đặt chỗ

1. Trong màn hình QL Đặt chỗ, nhấn nút **+ Tạo đặt chỗ** ở góc trên phải
2. Điền thông tin khách hàng

![Form tạo đặt chỗ](/img/vi/ql-dat-cho/02-tao-dat-cho.png)

## Thông tin cần điền

| Trường | Bắt buộc | Mô tả |
|--------|----------|-------|
| **Tên khách** | ✓ | Tên người đặt |
| **Số điện thoại** | ✓ | SĐT liên hệ |
| **Ngày** | ✓ | Ngày đến |
| **Giờ** | ✓ | Giờ đến dự kiến |
| **Số người** | ✓ | Số lượng khách |
| **Bàn** | | Chọn bàn cụ thể (tùy chọn) |
| **Ghi chú** | | Yêu cầu đặc biệt |
| **Đặt cọc** | | Số tiền cọc (nếu có) |

3. Nhấn **Lưu** để xác nhận đặt chỗ

## Sau khi tạo

Đặt chỗ mới sẽ xuất hiện trong danh sách với trạng thái **Chờ xác nhận**. Hệ thống có thể gửi SMS xác nhận cho khách (nếu được cấu hình).
