# Giao diện

Cài đặt **Giao diện** cho phép tùy chỉnh giao diện hiển thị của ứng dụng POS.

![Cài đặt giao diện](/img/vi/cai-dat/05-giao-dien.png)

## Các tùy chọn

| Cài đặt | Mô tả |
|---------|-------|
| **Chủ đề** | Sáng / Tối |
| **Bố cục menu** | Dạng lưới hoặc dạng danh sách |
| **Hiển thị ảnh món** | Bật/tắt ảnh minh họa trong menu |
| **Cỡ chữ** | Nhỏ / Vừa / Lớn |
| **Logo cửa hàng** | Upload logo hiển thị trên app |
| **Màu chủ đạo** | Tùy chỉnh màu nút, header |
